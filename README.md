# NOVA Development — лендинг застройщика

Премиальный лендинг жилого девелопера: тёмная тема с золотыми акцентами, планировки квартир с фильтром по комнатности, ход строительства с прогрессом по корпусам, ипотечный калькулятор.

**Демо:** https://trip-prog.github.io/nova-development-landing/

## Возможности

- Фильтр планировок по количеству комнат (студии — 3-комнатные)
- Ипотечный калькулятор: стоимость, первоначальный взнос, срок — расчёт ежемесячного платежа по ставке 4,7%
- Ход строительства с анимированными прогресс-барами
- Анимированные счётчики статистики, появление блоков при скролле
- Маска телефона и валидация в форме заявки
- Полностью адаптивная вёрстка, SVG-графика без внешних изображений

## Стек

HTML, CSS, ванильный JavaScript. Без сборщиков и зависимостей.

## Запуск

Открыть `index.html` или поднять любой статический сервер:

```
python -m http.server 8000
```
